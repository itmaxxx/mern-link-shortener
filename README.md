# Hi there :)

So, I created MERN app, following tutorial.

## Setup

1. Create `default.js` in `/config`
1. Put something like
`{
    "port": 5000,
    "mongoUri": "<YOUR_MONGODB_CONNECT_URL>",
    "jwtSecret": "<YOUR_SECRET>",
    "baseUrl": "http://localhost:5000"
}`
1. Start dev with `npm run dev`
